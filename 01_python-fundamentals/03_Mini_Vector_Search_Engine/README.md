# Mini Vector Search Engine

A beginner Python project that simulates how AI retrieval systems work — built while learning from the *Python for Everybody* specialization.

## How It Works

1. User enters a query
2. Query is split into words
3. Each document in the knowledge base is scored by word matches
4. The highest-scoring document is returned

**Example:**
```
Query:  What are embeddings
Output: Embeddings convert text into numerical vectors that capture semantic meaning.
```
