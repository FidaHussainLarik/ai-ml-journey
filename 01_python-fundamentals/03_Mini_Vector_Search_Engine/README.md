# Mini Vector Search Engine

## About Project

This is my mini project built while learning Python from the Python for Everybody specialization.
The idea of this project is to make a simple search engine that searches through a small knowledge base and finds the most relevant document based on the user's query.

I made this project to practice Python concepts like:

- Variables
- Lists
- Loops
- Functions
- Conditions
- String methods

and also to understand how AI systems search for information before giving answers.

## How It Works

1. User enters a query.
2. The query is converted into words.
3. The program checks every document in the knowledge base.
4. It counts how many words match between the query and document.
5. The document with the highest score is selected.
6. The best matching document is shown to the user.

## Example

**User Query:**
```
What are embeddings
```

**Output:**
```
Embeddings convert text into numerical vectors that capture semantic meaning.
```
